# 一统同济 (OneTJ)

「一统同济」是一个基于HarmonyOS开发的校园生活助手应用，旨在为同济大学学生提供便捷的校园服务。该应用集成了课程表管理、物理实验计算、校园资讯获取等功能，通过统一身份认证登录，为学生提供全方位的校园生活支持。

## 🚀 功能特性

### 1. 课程表管理
- **单日课表**：查看今日课程安排
- **数据同步**：自动获取学校课程信息
- **直观展示**：使用卡片形式展示课程详情

### 2. 物理实验计算
- **实验数据处理**：提供物理实验数据计算功能
- **玻尔扭摆实验**：支持用玻尔共振仪研究物体受迫振动
- **智能计算**：支持从学校实验表格中复制数据进行处理

### 3. 校园资讯
- **消息列表**：获取校园最新通知公告（公告数量并不完整）
- **详情查看**：支持查看详细消息内容
- **实时更新**：自动同步最新通知

### 4. 用户服务
- **统一身份认证**：使用同济大学统一身份登录
- **个人信息**：展示学生姓名、学号、学院等信息
- **学期信息**：显示当前学期和周次信息

### 5. 其他功能
- **应用设置**：个性化设置选项
- **关于我们**：应用信息和版本说明
- **加入讨论群**：快速加入用户交流群
- **隐私政策**：用户隐私保护说明

## 🛠 技术架构

- **开发平台**: HarmonyOS
- **开发语言**: ArkTS
- **UI框架**: ArkUI
- **数据存储**: 分布式KV存储
- **网络请求**: HTTP/HTTPS协议
